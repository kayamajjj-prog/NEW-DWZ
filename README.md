# Die With Zero プランナー

参照シートの値を**一度だけ反映**し、以後はアプリ内で都度修正するためのStreamlitアプリです。

## 起動方法
```bash
pip install -r requirements.txt
streamlit run app.py
```

起動後、ターミナルに表示されるURLから確認できます。
- Local URL: `http://localhost:8501`
- 同一ネットワーク共有時は Network URL が表示されます

## 使い方
1. まずは現在総資産と余暇予算比率を設定
2. 下の年次テーブルを直接編集（推奨）
3. 必要ならCSVを一度読み込み、以降はテーブルで修正
4. 年末資産・余暇予算の推移を確認

## ポイント
- 余暇予算 = 年初資産 × 比率（可変費）
- 必須収支が赤字の年は余暇予算を自動抑制
